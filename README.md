# eth-todo-list
- ToDo List demo created using. 
  - etheruem
  - solidity
  - web3.js
  - truffle
  - ganache
  - mocha.js
  - meta mask chrome extension
  
- dependances.
  - Truffle v5.4.26 (core: 5.4.26)
  - Solidity v0.5.16 (solc-js)
  - Web3.js v1.5.3
  
 # Commands
  - $ npm install -g truffle@5.4.26
  - $ truffle init
  - $ npm insatll
  - $ truffle compile
  - $ truffle migrate
  - $ truffle test
  - $ npm run dev
